## autolab

This project is a Python monorepo that defines research experiments under `experiments`.

Experiments implement the `autoresearch` interface defined in `src/autoresearch`. Each experiment is its own git-tracked project where agents can autonomously optimize research objectives.

The `autolab` project unifies experiment definition and the experiment harness in a single place, enabling collaboration and versioning.

See the `PROGRAM.md` file for additional experiment instructions.
